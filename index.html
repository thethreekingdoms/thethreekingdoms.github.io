什么是TTK？

  • TTK是一整套前端工程化解决方案，功能涵盖前端项目的起始、开发、构建、测试和部署上线阶段。
  • TTK提供可定制的脚手架、插件系统以及开发者API，你可以根据业务需求搭建自己的前端工程体系。

  • 解放生产力

    • 脚手架 / 环境配置
    • 规范测试 / 组件化开发
    • JS / Style / HTML / 图片

  • 流水线部署

    • 一键部署
    • 部署测试环境
    • 部署生成环境

  • 开发服务

    • 本地服务器
    • Mock服务
    • 多环境支持

  • 可定制系统

    • 插件系统
    • 定制脚手架
    • 高级配置
// 使用脚手架创建新的页面,webapp是自己定的项目名称
ttk new webapp
//进入当前目录
cd webapp
//启动web server
ttk start
打开浏览器访问 http://localhost:8082
